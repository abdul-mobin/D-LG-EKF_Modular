# Paper Recreation Modular

This repository contains MATLAB code for recreating and comparing radar tracking and beamforming experiments from the paper.

## Files
- Main_RADAR_Comparison.m: main entry point for running the comparison experiments.
- Supporting MATLAB functions for radar measurement generation, EKF/LGKF filtering, and plotting.

## Usage
1. Open MATLAB.
2. Run Main_RADAR_Comparison.m from the repository root.
